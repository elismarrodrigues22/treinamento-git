Quais os comandos para configurar usuário e email no git ? git config --global user.name "nome" git config --global eser.email "email"

O git é um sisema de controle de versão popular, é utilizado para
fazer alterações no código de acompanhamento, rastreamento de quem fez alterações
e colaboração de codificação

O github é uma plataforma baseada na web que incorpora os recursos de
controle de versões.

o que faz o comando commit ?
o commit é usado para confirmar as alterações preparadas .

o que faz o comando push?
envia o conteúdo do repositório local para um repositório remoto.

o que faz o comando pull?
é usado para buscar e baixar o conteúdo de repositórios remotos e fazer atualização imediata

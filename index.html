Qual a diferença entre o git e o github ?

O gité um sisema de controle de versão popular, é utilizado para
fazer alterações no código de acompanhamento, rastreamento de quem fez alterações
e colaboração de codificação

O github é uma plataforma baseada na web que incorpora os recursos de
controle de versões.
